= dotfiles

dotfiles I found handy.

== requirement

[source,bash]
----
sudo dnf install python3-pygments
----

== Structure

- link:bin/[bin/] contains binaries and will be add via `logical links` to the `/usr/local/bin/` folder
- link:.aliases[.aliases] - Shell aliases for common commands (Maven, Git, Kafka, etc.)
- link:.functions[.functions] - Custom bash functions
- link:.exports[.exports] - Environment variable exports
- link:.path[.path] - PATH configuration
- link:.jdk[.jdk] - JDK version management
- link:.bash_profile[.bash_profile] - Bash login profile
- link:.bashrc[.bashrc] - Bash configuration
- link:.bash_prompt[.bash_prompt] - Custom prompt configuration
- link:.gitconfig[.gitconfig] - Git configuration

== To install

[source,bash]
----
make
----

This will create symbolic links from the link:bin/[bin/] directory to `/usr/local/bin/` for all utility scripts.

== To customize

Save env vars, etc in a `.extra` file, that looks something like
this:

[source,bash]
----
###
### Git credentials
###

GIT_AUTHOR_NAME="Your Name"
GIT_COMMITTER_NAME="$GIT_AUTHOR_NAME"
git config --global user.name "$GIT_AUTHOR_NAME"
GIT_AUTHOR_EMAIL="email@you.com"
GIT_COMMITTER_EMAIL="$GIT_AUTHOR_EMAIL"
git config --global user.email "$GIT_AUTHOR_EMAIL"
GH_USER="nickname"
git config --global github.user "$GH_USER"

###
### Gmail credentials for mutt
###
export GMAIL=email@you.com
export GMAIL_NAME="Your Name"
export GMAIL_FROM=from-email@you.com
----

=== Change Jdk

To allow to run `sudo alternatives` with being promt for password - update /etc/sudos via

[source,bash]
----
sudo visudo
----

Add shomething like this to the bottom of the file

----
micbn   ALL=NOPASSWD:/usr/sbin/alternatives
----

- link:bin/git-squash[git-squash] - Squash git commits
